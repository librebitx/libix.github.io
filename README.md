# MyBlogSite

这是我的个人技术博客，用来记录学习笔记、实践经验和一些折腾过程。内容偏向 **Linux / 运维 / 容器 / 云原生 / 编程基础**，同时也会不定期更新工具使用心得和踩坑记录。

## ✨ 博客内容

博客目前主要包含以下几类内容：

- **Linux / 桌面环境**：发行版体验、桌面环境（GNOME / Openbox ）、系统配置等
-  **运维与系统管理**：服务器管理、性能分析、常用脚本
- **容器与云原生**：Docker、Kubernetes、Ansible、实验项目记录
- **编程与基础知识**：C 、Shell、后端基础
- **实验 & 折腾**：实验性项目、踩坑总结、对比测试

## 🛠 技术栈

本博客使用的技术栈可能会随着时间调整，目前包括但不限于：

- 静态站点生成：**Markdown + 静态站点生成器**
- 托管方式：**GitHub Pages**
- 主题：简洁、偏技术向，优先可读性
- 写作工具：本地编辑器 + Git
